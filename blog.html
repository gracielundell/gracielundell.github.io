The Basics of HTML/CSS

Jan 18, 2016

We all know that first impressions are vital to successful business and social interactions. In today’s society, webpages are increasingly becoming more influential as a means of presenting ourselves. If a webpage does not initially grab someone's attention, either through ease-of-layout, color scheme, etc, there is a large chance the user will leave the site sooner than they would have had it something that grabbed their attention. Even for people who do not care about visuals as much when it comes to the web, if a page is hard to maneuver they will not spend time there.

This is where well done HTML and CSS can come into play in vital ways. HTML (Hyper Text Markup Language) is used to place content on a webpage, using tags to display and manipulate different content. CSS (Cascading Style Sheets) is what is used to style the webpage on top of the HTML and determines where and how the content will lie on the page. Here are some examples of HTML:

            	HTML tags
              	<em>...</em>
              	<p>...</p>
              	<img>

The main difference between HTML and CSS is that HTMl tags are used with and when writing the actual webpage. For example, the

tag is used to begin a new paragraph, applied to the actual content of the page. The styling that is supplied by the CSS is done entirely in a separate file from the content and is linked to at the beginning of the page. As to targeting specific aspects of the page, CSS is referenced through classes, id's, and HTML tags. The CSS styling in its specific file may look something like this: 

              p {
                font-size: 20px;
                text-align: center;
              }

This exapmle of CSS styling targets the p tag (which tells the HTML where to add line breaks for paragraphs), and says that the text should be 20 pixels in size, and should be centered on the page.

Using only these two in conjunction, the HTML to determine the content and the CSS to style it, one can make a beautiful site, but will experience immense frustration in the process. But wait! The Bootstrap library, developed by the folks at Twitter, makes styling the webpage SO MUCH EASIER. One can apply already initialized classes inside a div tag that will automatically change the way the page looks. An example:

            	<div class=“jumbotron”>
            	</div>

This applied tag can then be (optionally) further styled to fit the individual’s specific taste. The most used tool for customizing Bootstrap is the Box Model.

The box model is used to segregate the different parts of the page that surround each HTML aspect. The HTML (content) is surrounds by three aspects: the margin, the border, and the padding.

Margin: transparent area between articles on the webpage. Provides spacing without affecting anything but layout on the page.

Border: surrounds the padding; may be partially or fully applied to an element on the webpage to add a visual separation (as with a line between sections on the page).

Padding: creates transparent space around the content on a page

Using the Bootstrap library in conjunction with the box model is an easy way to add specific styling to your webpage with minimal effort.
